## Introduction
Hey everyone, today I’ll show you how to build your own low cost BadUSB using Raspberry, similar to a Rubber Ducky, for just a few bucks

I’ve been experimenting with this for fun and created a version that mimics the functionality of the Hack5 Rubber Ducky. In this guide, I’ll walk you through the setup and how to get it running like a charm.


## <span style="color:#00ff88">Board</span>

For creating own BadUSB i'm using board Raspberry can checkout here [Pi Pico](https://my.cytron.io/p-raspberry-pi-pico-board)

## <span style="color:#00ff88">Installation Setup</span>

For the complete setup process and instructions on flashing the firmware and resetting the device, you can refer to the GitHub page for more [details](https://github.com/kacperbartocha/pico-badusb?tab=readme-ov-file)

Once you’ve successfully flashed the board, you can verify it by opening the device storage. You should see files such as:

- boot.py
- boot_out.txt
- main.py
- payload.txt
- license.txt
